# Data Types

- String (any charactor or text that wrap in single or double quotes)
    - Examples
        - `"pizza"`
        - `'burger'`
       

- Number (any numerical character)
    - Examples
        - `80/-`
        - `40/-`
- Boolean (similar to yes or no, 0 or 1)
    - Examples (not required quotes)
        - `true`
        - `false`
